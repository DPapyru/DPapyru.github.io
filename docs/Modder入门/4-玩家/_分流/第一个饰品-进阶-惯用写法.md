这份补充内容面向 **C≥1 且 T≥1**：默认你已能阅读本章代码，因此这里聚焦更惯用、可维护性更好的组织方式与注意点。

## 1）属性加成方法

`tModLoader` 提供了多种属性加成方法：

```csharp
player.GetDamage(DamageClass.Generic) += 0.1f;  // 伤害加成
player.GetCritChance(DamageClass.Generic) += 5;  // 暴击率加成
```

这些方法比直接修改 `damage` 更推荐使用。

## 2）条件激活的饰品

有些饰品效果只在特定条件下激活：

```csharp
if (!player.HasBuff(BuffID.Poisoned))
{
    // 只有没中毒时才生效
}
```

## 3）饰品槽位冲突处理

当多个饰品修改同一个属性时，tML 会处理叠加逻辑。了解这一点有助于设计合理的属性加成。
