# 第五课：方法与参数

到这里，你已经会写顺序、分支和循环。下一步就是“把重复逻辑提取出来”。这就是方法的作用。

## 本课目标

1. 定义并调用方法。
2. 使用参数传递数据。
3. 使用返回值拿到结果。
4. 通过方法复用减少重复代码。

## 什么是方法

方法是一段“可重复调用的代码块”。

如果你发现一段逻辑在多个地方都要写，通常就该提取为方法。

## 方法的基本结构

```csharp
返回类型 方法名(参数列表)
{
    // 方法体
    return 返回值;
}
```

当方法不需要返回结果时，返回类型用 `void`。

## 参数与返回值

- 参数：调用方传给方法的数据。
- 返回值：方法计算后交回给调用方的数据。

## 示例拆解

1. `PrintPlayerStatus` 用于输出状态，返回类型是 `void`。
2. `CalculateDamage` 接收基础伤害与加成，返回总伤害。
3. `IsHighDamage` 返回布尔值，用于后续分支判断。

## 验证练习

1. 把 `CalculateDamage(20, 5)` 改成你自己的值。
2. 新增一个方法 `GetPlayerTitle(int level)` 并返回称号。
3. 在 `Main` 中调用并输出称号。

## 常见错误

### 错误 1：方法签名与调用不一致

参数个数或类型不匹配会直接报错。

### 错误 2：有返回类型但忘记 `return`

编译器会提示并阻止编译。

## 小结

方法是后续面向对象与项目化开发的基础。你越早习惯拆方法，代码质量越高。
