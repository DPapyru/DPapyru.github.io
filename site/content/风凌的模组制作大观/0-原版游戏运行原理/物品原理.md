# 内容
一、Item 在泰拉瑞亚中是什么？

一句话定义：

Item 是一个“可被玩家持有、使用、丢弃、堆叠的状态对象”

它不是逻辑实体（不像 NPC / Projectile 那样每帧都有 AI），
只有在特定情况下才会参与逻辑更新。

二、Item 的2种存在形态（非常重要）

同一个 Item 类，在游戏里有2种完全不同的运行方式：

1️⃣ 玩家背包 / 饰品栏 / 手持物品（最常见）
存在于 Player.inventory[] 这个物品数组内
这种形式的物品只在特定方法执行时才参与逻辑，例如HoldItem,UpdateInInventory等

2️⃣ 世界中的掉落物（Item 实体）
存在于 Main.item[]这个物品数组内
Item.active = true
受重力、碰撞影响
可被拾取
这时它是一个“弱实体”
虽然没有AI(),但是有一个与AI()类似的UpdateInWorld()

当你把物品从背包中丢出时，实际上是在世界中的Main.item[]找一个active为false的空位，将你丢出去的物品属性复制给那个空位
于是世界中就有了你丢出去的物品的复制品，以掉落物形式出现。
而你背包中原本的那个物品会被代码删除，准确地说是隐去。
